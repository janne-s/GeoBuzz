# Example 1: Minimal Player

The simplest possible buzz player - perfect for learning the basics.

## Features

- Just map + play button
- Auto-centers on first sound
- Single button play/pause control

## Use Case

- Learning RuntimeEngine API
- Starting point for custom players
- Quick prototypes
- Understanding the essentials
