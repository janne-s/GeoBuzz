# Example 3: Spatial Visualization

Mapless buzz player with morphing canvas visualization. Sounds float in abstract space at their relative geographic positions, connected by evolving geometric patterns.

## Features

- Sounds positioned at relative geographic locations
- Morphing visual connections (Delaunay triangulation, particle systems, etc.)
- Full-screen immersive experience

## Visual Concepts

- **Delaunay triangles** connecting sound positions
- **Particle flows** between active sounds
- **Voronoi cells** pulsing with audio
- **Force-directed networks** of sound relationships

## Use Cases

- Art installations
- Live performances
- Ambient displays
- Gallery exhibitions
- Creative presentations
- Meditative listening experiences

## Technical

- Full-screen HTML5 Canvas
- Geographic coordinates mapped to screen space
- Delaunator.js for triangulation (or similar)
